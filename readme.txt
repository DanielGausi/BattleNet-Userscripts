Some Userscripts to improve/fix some links in the german BattleNet forums for Diablo 3.

Use them with Greasemonkey/Tampermonkey or whatever your Browser uses for Userscripts.

Content:


D3-deutsche-Profile:
--------------------
Change the link to a users profile to the german version in the german sub-forums



D3-Item-Guide-Wizard:
---------------------
Change the links in older guides on the official forums to the new link system in the Diablo 3 Game Guide
